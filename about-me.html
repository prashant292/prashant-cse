What Tools I Use

HTML 5

HTML5 is the latest version of the Hypertext Markup Language, enhancing web development with improved semantics, multimedia support, and dynamic graphics.

CSS 3

CSS3 is the third version of Cascading Style Sheets, a technology used for web page styling. It adds modern design capabilities like animations, gradients, and rounded corners to enhance the look and feel of websites.

BootStrap

Bootstrap is a widely-used open-source web framework that simplifies web development by providing pre-designed components and styles for building responsive and visually appealing websites and applications.

JavaScript

JavaScript is a versatile, high-level programming language commonly used for adding interactivity and dynamic behavior to websites. It runs in web browsers and can also be employed on the server-side for web applications.

TypeScript

Git (/ɡɪt/) is a distributed version control system that tracks changes in any set of computer files, usually used for coordinating work among programmers who are collaboratively developing source code during software development.

React JS

React JS, an open-source client-side scripting language, is essential for web development. It enables the creation of dynamic web pages and versatile web applications by seamlessly embedding client-side logic within HTML.

Next JS

Next.js is a React framework that enables server-side rendering and generating static websites. It is built on top of React and provides additional features such as: Server-side rendering (SSR), Static site generation (SSG), Automatic code splitting, Client-side routing, Built-in CSS and Sass support, API routing, Fully extendable.

GSAP

GSAP (GreenSock Animation Platform) is a powerful JavaScript animation library that enables you to create dynamic and eye-catching animations for your React application.

Locomotive

Locomotive Scroll is a JavaScript library that creates smooth scrolling effects on web pages. It enables programmers to produce visual effects where different elements on a page scroll at different rates to give the impression of depth and dimension.

Scroll Trigger

ScrollTrigger is a GSAP plugin that enables you to easily create animations based on the user's scroll position.

Figma

Figma is a browser-based web design tool that lets teams of users collaborate on projects to build interactive user interface prototypes. Users can modify texts, shapes, colors, and images and add functionalities such as hovering, scrolling, or animation to their interface assets.

Git

Java

Java is a platform-independent, object-oriented programming language known for its portability and versatility. It's used for developing a wide range of applications, from web and mobile apps to server-side software and embedded systems.

Python

Python is a high-level, versatile, and user-friendly programming language known for its readability and simplicity. It is widely used in web development, data analysis, artificial intelligence, and many other fields.
